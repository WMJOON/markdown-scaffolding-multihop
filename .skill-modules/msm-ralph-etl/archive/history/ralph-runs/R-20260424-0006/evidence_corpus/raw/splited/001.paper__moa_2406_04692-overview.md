# Abstract

본 논문은 다중 LLM의 집단적 전문성을 활용하는 새로운 접근 방식인 Mixture-of-Agents (MoA)를 제안합니다. MoA는 계층적(layered) 구조로 구성되며, 각 레이어의 에이전트는 이전 레이어의 출력을 보조 정보(auxiliary information)로 활용하여 응답을 생성합니다. 본 모델은 AlpacaEval 2.0, MT-Bench, FLASK에서 최신 성능(state-of-art performance)을 달성하며 GPT-4 Omni를 능가합니다. 특히, 오픈 소스 LLM만 사용한 MoA 모델은 AlpacaEval 2.0에서 65.1%라는 점수를 기록하며 GPT-4 Omni의 57.5%를 크게 앞지르는 선두를 차지했습니다.
