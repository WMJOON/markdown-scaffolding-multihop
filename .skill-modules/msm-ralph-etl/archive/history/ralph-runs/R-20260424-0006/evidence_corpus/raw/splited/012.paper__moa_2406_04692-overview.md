# 4 Related Work (1/2)

본 섹션은 LLM 성능 향상을 위한 두 가지 주요 접근 방식, 즉 추론 과정 최적화(LLM Reasoning)와 다중 모델 활용(Model Ensemble)을 다룹니다.

**LLM Reasoning:**
LLM의 생성 품질 향상을 위해 Prompt Engineering이 발전했으며, 기본적인 선형 문제 해결 접근법인 Chain of Thought (CoT) (Wei et al., 2022; Kojima et al., 2022)부터 시작했습니다. 추론 과정의 복잡성을 높인 기법으로는 여러 경로를 고려하고 자기 평가를 수행하는 Tree-of-Thought (ToT) (Yao et al., 2023a)와 생각의 그래프로 구조화한 Effective Graph-of-Thought (Yao et al., 2023b)가 있습니다. 또한, 과제 분해(PS Prompt by Wang et al., 2023)나 불확실한 질문 선택(Active-Prompt by Diao et al., 2023) 등 다양한 방식으로 CoT를 자동화하고 개선했습니다.

**Model Ensemble:**
모델 앙상블 분야에서는 여러 모델의 강점을 결합하는 방식이 연구되었는데, 간단한 방식으로는 PAIRRANKER (Jiang et al., 2023)와 같이 후보 출력 간 쌍별 비교를 통한 재순위 지정(reranking)이 있습니다. 비용 절감을 위해서는 라우터 학습(Wang et al., 2024a; Shnitzer et al., 2024; Lu et al., 2023)이나 캐스케이딩 방식(FrugalGPT by Chen et al., 2023b)을 활용합니다. 모델 간의 응답을 결합하는 방법으로는 여러 후보를 이용해 개선된 응답을 생성하는 GENFUSER (Jiang et al., 2023)나 출력 확률 분포 평균화(Huang et al., 2024)가 있습니다. 가장 복잡한 형태는 다중 에이전트 협업으로, Du et al. (2023)의 대칭적 논의나 MAD (Liang et al., 2023)의 토론가(debater)와 심판(judge) 역할을 포함하는 비대칭적 메커니즘 등이 활용됩니다.
