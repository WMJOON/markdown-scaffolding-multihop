# 2 Mixture-of-Agents Methodology (2/8)

본 논문에서 제안하는 MoA(Mixture-of-Agents) 방법론은 여러 LLM $A_{i,j}$가 생성한 다양한 출력을 통합하여 고품질의 응답을 생성하는 프레임워크입니다. MoA는 입력 $x_1$에 대해 $i$-번째 MoA 레이어의 출력 $y_i$를 $y_i = \bigoplus_{j=1}^{n} [A_{i,j}(x_i)] + x_{i+1}$로 정의하며, 여기서 $\bigoplus$ 연산은 Table 1의 "Aggregate-and-Synthesize Prompt"를 사용하여 여러 모델 출력을 종합적으로 합성하는 과정을 의미합니다. MoA는 기존의 MoE(Mixture-of-Experts) 방법론(Shazeer et al., 2017)에서 영감을 받았으나, MoE가 활성화(activation) 수준에서 동작하는 것과 달리 MoA는 LLM 자체의 출력을 활용하여 '모델 수준'에서 작동한다는 것이 핵심 차이점입니다. 실제 구현 시에는 마지막 레이어의 $A_{l,1}(x_l)$ 출력을 최종 결과로 사용하며, 합성 과정은 응답의 정확성과 포괄성을 보장하는 방식으로 이루어집니다.
