# 2 Mixture-of-Agents Methodology (3/8)

제시된 내용을 바탕으로 핵심 3-5문장 요약은 다음과 같습니다.

MoA는 내부 가중치나 활성화 값 변경 없이 프롬프트 인터페이스만을 이용해 작동하는 새로운 Methodology입니다. 이 접근법은 MoE와 달리 여러 개의 완전한 LLM을 계층별로 활용하며, 게이팅 네트워크와 전문가 네트워크의 역할을 LLM 자체가 수행하여 외부 조정 메커니즘이 필요 없습니다. 이 방식은 파인튜닝에 따른 계산 오버헤드를 제거하고, 모델 크기나 아키텍처에 관계없이 최신 LLM에 적용할 수 있는 높은 유연성과 확장성을 제공합니다. 평가 결과, MoA는 AlpacaEval 2.0, MT-Bench, FLASK 벤치마크에서 개선된 성능을 보여주었으며, 특히 오픈소스 모델만으로 GPT-4o를 능가하는 결과를 달성했습니다. 또한, 상세한 예산 분석을 통해 MoA의 여러 구현체들이 GPT-4 Turbo와 비교하여 비슷한 성능을 제공하면서도 비용 효율성이 2배 더 높은 것으로 나타났습니다.
