# References (2/4)

이 참고문헌 목록은 대규모 언어 모델(LLM)의 구조적 개선, 고도화된 추론 능력 구현, 그리고 다양한 응용 모델의 출현 등 폭넓은 최신 연구 동향을 보여줍니다.

LLM의 능력 향상을 위해, 모델은 zero-shot reasoner로서의 능력을 입증하고(Kojima et al., 2022), 복잡한 추론 과정을 지원하기 위해 인간 피드백 기반 훈련(Ouyang et al., 2022) 및 chain-of-thought의 연역적 검증(Ling et al., 2023) 방법론이 활용됩니다. 추론 능력 자체는 다중 에이전트 토론(Liang et al., 2023)이나 pairwise ranking 및 generative fusion(Rogers et al., 2023) 같은 고급 방법을 통해 강화되고 있습니다. 구조적 측면에서는 효율적인 운영을 위해 Sparse-gated mixture-of-experts layer(Shazeer et al., 2017) 개념과 같은 희소성을 이용한 모델 라우팅(Lu et al., 2023; Shnitzer et al., 2024) 연구가 핵심적입니다. 이러한 기술적 진보를 바탕으로, Gemini(Team et al., 2023)와 같이 멀티모달 기능을 통합한 최신 모델들이 출현하고 있으며, Code llama(Roziere et al., 2023)나 dbrx(2024)와 같은 특정 도메인에 특화된 개방형(open) 모델들이 활발히 개발되고 있습니다.
