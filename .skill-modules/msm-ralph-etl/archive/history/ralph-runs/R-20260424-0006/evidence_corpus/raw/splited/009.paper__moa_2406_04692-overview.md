# 2 Mixture-of-Agents Methodology (6/8)

본 내용은 Mixture-of-Agents (MoA) 방법론의 성능을 입증하는 실험 결과를 다룹니다. 분석 결과, 텍스트 유사성을 측정하는 다른 접근법들도 preference scores와 양의 상관관계(positive correlation)를 보였습니다.

성능에 영향을 미치는 주요 변수로, 에이전트의 수($n$)는 더 많은 보조 정보 덕분에 $n$ 값이 증가함에 따라 점수가 단조롭게 증가하는 경향을 보였습니다. 또한, 결과의 품질 향상 측면에서 "multiple-proposer" 방식은 "single-proposer" 방식보다 일관되게 더 나은 결과를 제공했습니다.

모델별 성능 비교(Table 4)를 통해, Qwen1.5-110B-Chat과 같은 모델은 에이그게게이터로 사용될 때와 프로포저로 사용될 때의 성능 차이를 보여주었으며, 예를 들어 WizardLM 8x22B는 에이그게게이터로서 52.9%의 성능을 보인 반면, 프로포저로서 63.8%를 기록했습니다.

마지막으로, 평가 과정에서 GPT-4o 모델은 오직 출력을 집계(aggregate)하는 목적으로만 사용되었으며, Proposer로 참여하지 않았음을 명시했습니다. 전반적으로, 다양한 LLM 에이전트를 MoA 레이어에 사용하는 것이 성능 향상에 기여하는 것으로 나타났습니다.
